Home Assistant Crypto Profit/Loss Sensor Code Creator

Because I got tired of re-learning how to code this thing every. single. time. I bought a new coin

If you run Home Assistant and you own crypto, this handy bit of kit will generate code to give you a sensor card like the ones above. It will tell you one thing: How much am I making or losing on that crypto I bought? Once installed in your Homeassistant configuration file, it uses the free Coingecko API (thanks, Coingecko!) to fetch the current price of the coin, then does a little math comparing the price you paid and the number of coins you bought. The result is an at-a-glancee picture of how your Lambo plans are going. And if your Lambo plans are proceeding nicely, and you find this useful, buy me a coffee. Or send me some Eth or Bitcoin.
